# Global Superstore Sales Analysis Dashboard

## Project Overview
This project analyzes Global Superstore sales data to understand sales performance, profit trends, and regional business insights using data visualization techniques.

## Dataset
The analysis is based on the Global Superstore dataset containing sales, profit, category, and regional information.

## Dashboard Features
- Sales performance analysis
- Profit trend visualization
- Regional and category comparison
- KPI-based dashboard insights

## Tools Used
- Power BI (Dashboard Design)
- Excel (Dataset)
- Data Visualization
- Data Analysis
